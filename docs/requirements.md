# REQUIREMENTS.md

## Functional
- Display map with trip locations
- Collect voice messages from telegram
- Analyze emotions in voice message
- Match locations with voice messages timestamp
- Store matched data in DB

## Non-Functional
- Responsive web UI
- Fast map rendering

## User Stories
- As a user, I want to see my trip on a map with emotions per location
- As a user, I want to see new voice messages and see them matched
